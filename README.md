# 🚗 Scroll-Based Car Animation

This project is a scroll-based animation built using GSAP and ScrollTrigger.  
As the user scrolls, a car moves from left to right, revealing text and animated cards.

---

## ✨ Features

- Smooth scroll-based car movement
- Letter-by-letter text reveal ("WELCOME ITZFIZZ")
- Animated trail effect behind the car
- 4 info cards appearing during scroll
- Responsive and clean UI

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- GSAP (ScrollTrigger)
